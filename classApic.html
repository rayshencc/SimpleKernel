Detailed Description

APIC 管理类，管理整个系统的 Local APIC 和 IO APIC.

Copyright
Copyright The SimpleKernel Contributors
Note
在多核系统中：
  • Local APIC 是 per-CPU 的，每个核心通过 MSR 访问自己的 Local APIC
  • IO APIC 是系统级别的，通常有 1-2 个，处理外部中断
