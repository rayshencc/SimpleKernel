Detailed Description

先来先服务 (FIFO) 调度器

Copyright
Copyright The SimpleKernel Contributors

FIFO 调度器特点：

  • 先进先出，适合对延迟敏感的实时任务
  • 不可抢占，任务运行直到主动让出 CPU
  • O(1) 时间复杂度
