Detailed Description

互斥锁（Mutex）

Copyright
Copyright The SimpleKernel Contributors

实现基于任务调度的互斥锁：

  • 当锁被占用时，请求线程会被阻塞并进入等待队列
  • 当锁释放时，会唤醒一个等待线程
  • 支持所有者跟踪和递归检测
Note
使用限制：
  1. 不可重入：同一线程不能递归获取同一锁
  2. 所有权：必须由获取锁的线程释放锁
  3. 阻塞语义：获取锁失败时会阻塞当前任务
  4. 必须在任务上下文中使用：不能在中断处理程序中使用
